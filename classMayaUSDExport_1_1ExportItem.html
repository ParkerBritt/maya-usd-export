Detailed Description

Represents a single Maya DAG node intended for export to USD.

ExportItem encapsulates the essential data required to export a Maya object to USD. It stores the DAG path of the Maya object and the corresponding USD primitive type that it should be represented as.

Note
Future implementations may include export arguments for finer control.
